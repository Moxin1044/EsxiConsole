# 前言
EsxiConsole(Ex)是一款以Python为基础的Esxi控制器，我们致力于编写一个方便拓展的、功能简洁的ESXI控制器。

目的是以更简洁的方式接入ESXI，并且对多平台支持，适合开发使用，减少使用臃肿的ESXI Web。
